Data Privacy Pledge
Code is scanned in-memory and deleted immediately. We never store your source code. SOC2 Type 1 Compliant.
2. Add to GitHub Actions
name: Quantum Shield Scan
on: [push]
jobs:
  scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: |
          zip -r code.zip . -x "*.git*"
          curl -X POST "https://quantum-shield.onrender.com/scan" \
            -H "x-api-key: ${{ secrets.QS_API_KEY }}" \
            -F "file=@code.zip"
